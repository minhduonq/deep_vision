## Hướng dẫn cài đặt và sử dụng
Để chạy ứng dụng, cần cài đặt các thư viện và thêm các file cần thiết.

chạy Backend:
```
cd backend
pip install -r requirements.txt
python api/main.py
```

chạy frontend:
```
cd frontend-react
npm install
npm run dev
```

Để chạy được model NanoBanana, cần có API Replicate. Cấu hình nó trong file <code>.env</code>
